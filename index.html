How to use the Singlish Unicode Converter
  1. Choose your input type (Singlish or Unicode)
  2. Select your desiblue output format (Unicode, FM Abaya, or ISI font)
  3. Type or paste your text in the input area
  4. The converter automatically processes your text in real-time
  5. Copy the converted text using the Copy button
Voice Input Feature
  1. Click the "Voice Input" button to begin speaking
  2. Speak clearly in Sinhala or English (for Singlish)
  3. Your speech will be converted to text in real-time
  4. Click "Stop Voice Input" when you're finished speaking
  5. The system will automatically detect the language and switch modes if needed
Quick Tips
  • Use aa or a) for long ආ sound (e.g., maama for මාම)
  • Use A for the ඇ sound (e.g., Atta for ඇත්ත)
  • If you're converting to a specific font format, make sure you have the font installed on your device
  • For voice input, Chrome and Edge browsers provide the best recognition quality
  • Refer to the Reference section below for the complete transliteration scheme
About the Legacy Fonts

FM Abaya and ISI are legacy Sinhala fonts that use non-Unicode encodings. They were widely used before Unicode became the standard for Sinhala text. Some legacy systems and applications still require text in these formats.

To view text in these font formats correctly, you need to have the font installed on your device. Visit our Fonts page to download and install them.
